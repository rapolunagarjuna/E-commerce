# E-commerce website
An E-commerce website for par-global.

Features:
 - Secured User Authentication.
 - OCR Integration to automate scanned PO's insertion into database.
 - 
 - Containerized the application.

client development:
    cd client
    npm i
    npm run dev

backend development:
    cd backend
    npm i
    node app.js

docker command:
    docker-compose up --build
